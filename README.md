# NumPy Analyzer

A console-based **Python Data Analysis Toolkit** built using **NumPy and Object-Oriented Programming (OOP)**.
This project allows users to perform common data operations such as array creation, mathematical computations, searching, sorting, filtering, and statistical analysis.

---

## 🚀 Project Objective

The goal of this project is to build a **NumPy Analyzer** that integrates:

* NumPy functionalities
* Object-Oriented Programming concepts
* Menu-driven user interface

The toolkit allows users to perform **data operations and statistical analysis on datasets using NumPy arrays**.

---

# 🧠 Features

### Array Management

* Create **1D, 2D, and 3D NumPy arrays**
* Perform **indexing and slicing**
* Access rows, columns, and array segments

### Mathematical Operations

* Addition
* Subtraction
* Multiplication
* Division
* Dot product and matrix multiplication

### Combine & Split Arrays

* Combine arrays using **vertical stacking**
* Split arrays into smaller arrays

### Search, Sort & Filter

* Search values inside arrays
* Sort arrays in ascending order
* Filter arrays based on conditions

### Aggregation Functions

* Sum
* Mean
* Median
* Standard Deviation
* Variance

### Statistical Functions

* Minimum value
* Maximum value
* Percentiles
* Correlation between arrays
